# Workflow Template

Use this when writing a repeatable multi-step work sequence.

```markdown
# <Domain> <Action> Workflow

## Trigger

Use this workflow when:

- <trigger>

## Inputs

- <input>

## Process

1. <step>
2. <step>
3. <step>

## Outputs

- <artifact path and format>

## Quality Checks

- <check>

## Stop Conditions

- <condition>

## Completion Note

Report:

- changed files
- validation run
- skipped checks and why
- remaining risks
```
